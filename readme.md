# Windows Calculator Clone

![Calculator image](https://github.com/JulioMissart/Calculator-Java/blob/master/calc.png)

This project was made completely from scratch, based on a course on graphical user interfaces in java that shows the different types of components and how to use them.

# Features

As a standar calculator, you can:
+ Add
+ Substract
+ Multiply
+ Divide
+ One divided by X
+ Get square root
+ Mod
+ Raise
